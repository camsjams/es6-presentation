Object.assign()

Extending things with other things into potentially new things

jQuery


                        var o1 = { a: 1 };
                        var o2 = { b: 2 };
                        // clone
                        var copy = $.extend({}, o1);
                        // merge
                        var obj = $.extend(o1, o2);

Underscore


                        var o1 = { a: 1 };
                        var o2 = { b: 2 };
                        // clone
                        var copy = _.extend({}, o1);
                        // merge
                        var obj = _.extend({}, o2);

ES6


                     var o1 = { a: 1 };
                     var o2 = { b: 2 };
                     // clone
                     var copy = Object.assign({}, o1);
                     // merge
                     var obj = Object.assign(o1, o2);

Classes

Do you even prototype, bro?

ES6

        class Person {

            constructor(name) {
                this.name = name;
            }

            getInfo() {
                return `My name is ${this.name}`
            }
        }
                
        let Cam = new Person('Cameron');
        Cam.getInfo();    // "My name is Cameron"

Inheritance

Hierarchical hell, made easy!

ES6

        class Jerk extends Person {

            constructor(name) {
                super(name);
                this.jerk = true;
            }

            isJerk() {
                return this.jerk;
            }
        }
                
        let Justin = new Jerk();
        Justin.getInfo();    // "My name is Justin"
        Justin.isJerk();    // true

Default Parameters

So very convenient

ES6

            function f(x, y = 7, z = 42) {
                return x + y + z
            }
ES5

            function f (x, y, z) {
                if (y === undefined)

                if (z === undefined)
                    z = 42;
